## Security: ❌ CRITICAL FAIL (Grade: F)

### Shell Injection Vulnerabilities (CRITICAL)

**File:** `packages/demo-toolkit/src/tts.ts:25-27`
```typescript
await execAsync(
  \`openclaw tts --text "\${text.replace(/"/g, '\\"')}" --output "\${outputPath}"\`
);
```

**Vulnerability:** Insufficient escaping. Attack vectors:
- \`text='"; rm -rf /; echo "'\` - command injection
- \`outputPath='/tmp/x"; cat /etc/passwd;'\` - path injection
- \`text='Hello \$(whoami)'\` - command substitution

**Impact:** Arbitrary command execution with Node.js process permissions.

**Similar vulnerabilities in:**
- \`getAudioDuration()\` line 66 - audioPath unescaped
- \`combineVideoAudio()\` line 104 - all params unescaped
- \`createAudioTimeline()\` line 126 - clip paths unescaped

**REQUIRED FIX:** Replace shell strings with \`spawn()\` + argument arrays.
