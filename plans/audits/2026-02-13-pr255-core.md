## Executive Summary

This PR fixed a legitimate bug where HTMLElement values in JSX expressions were being stringified to `[object HTMLElement]` instead of being appended as actual DOM nodes. The technical solution is sound, introducing `__child()` and `__insert()` helpers with comprehensive test coverage.

However, the PR suffers from **multiple major process violations**: missing changeset, significant scope creep (bundled unrelated onMount/onCleanup fix), merged with failing CI, and likely TDD violations based on commit message ordering.
