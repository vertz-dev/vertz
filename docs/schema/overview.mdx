# Schema

The Vertz schema system provides type-safe data modeling.

## Overview

Define your data once, get type-safe code everywhere:

- **Entity definitions** — Type-safe data models
- **Relations** — Connect entities together
- **Validation** — Built-in data validation
- **Migrations** — Schema versioning

## Quick Example

```typescript
import { domain } from 'vertz/schema';

const blog = domain({
  name: 'blog',
  entities: {
    User: {
      id: { type: 'uuid', primary: true },
      email: { type: 'string', unique: true },
      name: { type: 'string' },
    },
  },
});
```
