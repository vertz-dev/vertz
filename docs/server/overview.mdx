# Server

The Vertz server handles SSR, API routes, and server functions.

## Overview

- **SSR** — Server-side rendering for fast initial loads
- **Streaming** — Progressive hydration
- **API Routes** — Type-safe endpoints
- **Server Functions** — Call backend from UI

## Quick Example

```typescript
import { createServer } from 'vertz/server';

const server = createServer({
  domains: [blogDomain],
  port: 3000,
});

server.start();
```
