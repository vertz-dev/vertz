# Primitives API

Detailed API reference for each primitive in `@vertz/primitives`.

## Button

Accessible button with press state.

```tsx
import { Button } from '@vertz/primitives';

const { root, state } = Button.Root({
  disabled: false,
  onPress: () => console.log('pressed!'),
});

root.textContent = 'Click me';
document.body.appendChild(root);
```

**API:**

```typescript
Button.Root(options: ButtonOptions): ButtonElements & { state: ButtonState }

interface ButtonOptions {
  disabled?: boolean;
  onPress?: () => void;
}

interface ButtonState {
  disabled: Signal<boolean>;
  pressed: Signal<boolean>;
}

interface ButtonElements {
  root: HTMLButtonElement;
}
```

## Checkbox

Checkbox with indeterminate state support.

```tsx
import { Checkbox } from '@vertz/primitives';

let isChecked = false;

const { root, state } = Checkbox.Root({
  checked: isChecked,
  onCheckedChange: (checked) => {
    isChecked = checked === true;
  },
});
```

**API:**

```typescript
interface CheckboxOptions {
  checked?: boolean | 'indeterminate';
  onCheckedChange?: (checked: boolean | 'indeterminate') => void;
}

interface CheckboxState {
  checked: Signal<boolean | 'indeterminate'>;
  disabled: Signal<boolean>;
}

interface CheckboxElements {
  root: HTMLInputElement;
}
```

## Dialog

Modal dialog with focus trap and backdrop.

```tsx
import { Dialog } from '@vertz/primitives';

const { overlay, content, state } = Dialog.Root({
  open: false,
  onOpenChange: (open) => console.log(open),
});

content.innerHTML = '<h2>Title</h2><p>Content</p>';
```

**API:**

```typescript
interface DialogOptions {
  open?: boolean;
  onOpenChange?: (open: boolean) => void;
}

interface DialogState {
  open: Signal<boolean>;
}

interface DialogElements {
  overlay: HTMLDivElement;
  content: HTMLDivElement;
}
```

The dialog handles:
- Focus trapping (Tab cycles through focusable elements)
- Escape key to close
- Click outside overlay to close

## Select

Single-select dropdown.

```tsx
import { Select } from '@vertz/primitives';

const { trigger, content, state } = Select.Root({
  value: 'en',
  onValueChange: (value) => console.log(value),
});

const en = Select.Option({ value: 'en' });
en.textContent = 'English';
content.appendChild(en);
```

**API:**

```typescript
interface SelectOptions {
  value?: string;
  onValueChange?: (value: string) => void;
}

interface SelectState {
  value: Signal<string>;
  open: Signal<boolean>;
}

interface SelectElements {
  trigger: HTMLButtonElement;
  content: HTMLDivElement;
}

Select.Option(options: { value: string }): HTMLDivElement
```

## Menu

Dropdown menu with arrow key navigation.

```tsx
import { Menu } from '@vertz/primitives';

const { trigger, content, state } = Menu.Root({
  open: false,
  onOpenChange: (open) => console.log(open),
});

trigger.textContent = 'Menu';

const item1 = Menu.Item({ onSelect: () => console.log('edit') });
item1.textContent = 'Edit';
content.appendChild(item1);
```

**API:**

```typescript
interface MenuOptions {
  open?: boolean;
  onOpenChange?: (open: boolean) => void;
}

interface MenuState {
  open: Signal<boolean>;
}

interface MenuElements {
  trigger: HTMLButtonElement;
  content: HTMLDivElement;
}

Menu.Item(options: { onSelect?: () => void }): HTMLDivElement
Menu.Separator(): HTMLDivElement
```

Keyboard: Arrow Up/Down to navigate, Enter/Space to select, Escape to close.

## Tabs

Tabbed interface with keyboard navigation.

```tsx
import { Tabs } from '@vertz/primitives';

const { list, state } = Tabs.Root({
  defaultValue: 'general',
  onValueChange: (value) => console.log(value),
});

const tab1 = Tabs.Trigger({ value: 'general' });
tab1.textContent = 'General';

const panel1 = Tabs.Content({ value: 'general' });
panel1.textContent = 'General settings...';

list.appendChild(tab1);
```

**API:**

```typescript
interface TabsOptions {
  defaultValue?: string;
  onValueChange?: (value: string) => void;
}

interface TabsState {
  value: Signal<string>;
}

interface TabsElements {
  list: HTMLDivElement;
}

Tabs.Trigger(options: { value: string }): HTMLButtonElement
Tabs.Content(options: { value: string }): HTMLDivElement
```

Keyboard: Arrow Left/Right to navigate tabs, Home/End to jump.

## Accordion

Collapsible sections.

```tsx
import { Accordion } from '@vertz/primitives';

const { root, state } = Accordion.Root({
  collapsible: true,
  multiple: false,
});

const item1 = Accordion.Item({ value: 'item-1' });
const header1 = Accordion.Header({ value: 'item-1' });
const trigger1 = Accordion.Trigger({ value: 'item-1' });
trigger1.textContent = 'Section 1';

const content1 = Accordion.Content({ value: 'item-1' });
content1.textContent = 'Content here';
```

**API:**

```typescript
interface AccordionOptions {
  collapsible?: boolean;
  multiple?: boolean;
}

Accordion.Root(options: AccordionOptions): { root: HTMLDivElement; state: Signal<...> }
Accordion.Item(options: { value: string }): HTMLDivElement
Accordion.Header(options: { value: string }): HTMLDivElement
Accordion.Trigger(options: { value: string }): HTMLButtonElement
Accordion.Content(options: { value: string }): HTMLDivElement
```

## Slider

Range input with thumb and track.

```tsx
import { Slider } from '@vertz/primitives';

let value = 50;

const { root, state } = Slider.Root({
  value,
  min: 0,
  max: 100,
  step: 1,
  onValueChange: (v) => { value = v; },
});
```

**API:**

```typescript
interface SliderOptions {
  value?: number;
  min?: number;
  max?: number;
  step?: number;
  disabled?: boolean;
  onValueChange?: (value: number) => void;
}

interface SliderState {
  value: Signal<number>;
  disabled: Signal<boolean>;
}
```

Keyboard: Arrow Left/Right to adjust, Home/End for min/max.

## Toast

Notification toast with auto-dismiss.

```tsx
import { Toast } from '@vertz/primitives';

const { root, state } = Toast.Root();

Toast.Toast({
  title: 'Saved',
  description: 'Your changes have been saved.',
  duration: 3000,
});
```

**API:**

```typescript
Toast.Root(): { root: HTMLDivElement }
Toast.Toast(options: ToastOptions): void

interface ToastOptions {
  title?: string;
  description?: string;
  duration?: number;
}
```

## Utilities

Import utilities from `@vertz/primitives/utils`:

```typescript
import {
  uniqueId,
  setLabelledBy,
  setDescribedBy,
  getFocusableElements,
  trapFocus,
  handleListNavigation,
  Keys,
} from '@vertz/primitives/utils';
```

**ARIA Helpers:**
- `setLabelledBy(el, id)` — Set `aria-labelledby`
- `setDescribedBy(el, id)` — Set `aria-describedby`
- `setExpanded(el, expanded)` — Set `aria-expanded`
- `setDisabled(el, disabled)` — Set `aria-disabled`

**Focus:**
- `getFocusableElements(container)` — Get all focusable elements
- `trapFocus(container)` — Trap focus within container
- `saveFocus()` — Save/restore focus

**Keyboard:**
- `handleActivation(event, callback)` — Enter/Space
- `handleListNavigation(event, items, currentIndex, onChange)` — Arrow keys
- `Keys` — Key constants (`Keys.Enter`, `Keys.Escape`, etc.)

**IDs:**
- `uniqueId(prefix?)` — Generate unique ID
- `resetIdCounter()` — Reset for testing
