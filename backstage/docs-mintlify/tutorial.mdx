# Full Stack Tutorial

Welcome to the Vertz full-stack tutorial! In this guide, you'll build a complete LLM-powered application from scratch.

## What You'll Build

A **customer support chatbot** that:
- Has a type-safe React UI
- Connects to an LLM for intelligent responses
- Uses a database for conversation history
- Implements authentication and authorization

## Prerequisites

- Node.js 18+
- npm, yarn, or pnpm
- An OpenAI API key (or compatible LLM)

## Tutorial Sections

1. [Project Setup](/tutorial/setup) — Create your first Vertz project
2. [Database & Entities](/tutorial/database) — Set up your data layer
3. [Authentication](/tutorial/authentication) — Add user login
4. [Building the API](/tutorial/api) — Create type-safe server procedures
5. [Building the UI](/tutorial/ui) — Create the chat interface
6. [Connecting to LLM](/tutorial/llm) — Add AI capabilities
7. [Deployment](/tutorial/deployment) — Ship to production

## Estimated Time

**45 minutes** to complete all sections.
