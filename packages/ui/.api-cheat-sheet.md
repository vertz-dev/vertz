# @vertz/ui API Cheat Sheet

## Core Exports (from index.ts)

### Reactivity Runtime
- `signal<T>(initial: T): Signal<T>` - Create reactive value
- `computed<T>(fn: () => T): Computed<T>` - Create derived value
- `effect(fn: () => void): DisposeFn` - Run side effect on changes
- `batch(fn: () => void): void` - Group updates
- `untrack<T>(fn: () => T): T` - Read without subscribing
### Component Lifecycle
- `onMount(fn: () => (() => void) | void): void` - Run once on mount (return a function for cleanup)
- `watch<T>(dep: () => T, callback: (value: T) => void): void` - Watch dependency

### Component Utilities
- `createContext<T>(defaultValue?: T): Context<T>` - Create context
- `useContext<T>(ctx: Context<T>): T | undefined` - Read context
- `children(accessor: ChildrenAccessor): () => Node[]` - Resolve children
- `ref<T>(): Ref<T>` - Create ref for DOM access
- `ErrorBoundary({ children, fallback })` - Catch errors
- `Suspense({ children, fallback })` - Handle async

### CSS & Theming
- `css<T>(styles: T): CSSOutput<T>` - Scoped styles (returns flat class name properties, e.g., `styles.card`)
- `variants(config: VariantsConfig): VariantFunction` - Typed variants
- `defineTheme(theme: ThemeInput): Theme` - Define theme
- `compileTheme(theme: Theme): CompiledTheme` - Compile theme
- `ThemeProvider({ theme, children })` - Provide theme
- `globalCss(styles: GlobalCSSInput): void` - Global styles
- `s(styles: CSSInput): string` - Inline dynamic styles

### Forms
- `form<TBody, TResult>(sdkMethod, options): FormInstance` - Create form
- `formDataToObject(formData, options?): Record<string, unknown>` - Convert FormData
- `validate<T>(schema, data): ValidationResult<T>` - Validate data

### Data Fetching
- `query<T>(thunk: () => Promise<T>, options?): QueryResult<T>` - Reactive fetch

### Router
- `defineRoutes<const T>(map: T): TypedRoutes<T>` - Define routes (preserves literal path keys)
- `createRouter<T>(routes: TypedRoutes<T>, initialUrl?): Router<T>` - Create router (typed navigate)
- `useRouter<T>(): Router<T>` - Access router from context (typed when parameterized)
- `useParams<TPath>(): ExtractParams<TPath>` - Typed route params accessor
- `InferRouteMap<T>` - Extract route map type from `TypedRoutes<T>`
- `createLink(currentPath, navigate): (props: LinkProps) => HTMLAnchorElement` - Create Link
- `createOutlet(outletCtx: Context<OutletContext>): () => Node` - Create outlet
- `parseSearchParams<T>(urlParams, schema?): T` - Parse search params
- `useSearchParams(searchParams)` - Use search params

### Hydration
- `hydrate(registry)` - Hydrate components (automatic strategy via IntersectionObserver)

## Signal API Details

### Signal<T>
```ts
signal.value       // Read (subscribes if in tracking context)
signal.value = x   // Write
signal.peek()      // Read without subscribing
signal.notify()    // Manually notify subscribers
```

### Computed<T> (ReadonlySignal)
```ts
computed.value     // Read (lazy, cached)
computed.peek()    // Read without subscribing
```

## Key Types
- `Signal<T>` - Read/write reactive value
- `ReadonlySignal<T>` - Read-only signal
- `Computed<T>` - Derived signal
- `DisposeFn` - `() => void` cleanup function
