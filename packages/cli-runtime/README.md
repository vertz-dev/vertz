# @vertz/cli-runtime

Runtime for CLIs generated by `vertz codegen`. Handles command parsing, help generation, parameter resolution, authentication, and formatted output.

## How it works

When you run `vertz codegen` with CLI generation enabled, it generates a CLI for your API that depends on this runtime:

```
vertz codegen вҶ’ generates CLI manifest + entry point
                    вҶ“
             @vertz/cli-runtime powers the generated CLI at runtime
```

Your generated CLI gets commands like `myapp users list`, `myapp users get --id 123` вҖ” all fully typed and wired to your API.

## Quick Start

```typescript
import { createCLI } from '@vertz/cli-runtime';

const cli = createCLI(manifest, {
  baseURL: 'https://api.example.com',
});

await cli.run(process.argv.slice(2));
```

The `manifest` is auto-generated by `@vertz/codegen` вҖ” you don't write it by hand.

## What the runtime provides

### Command parsing

Extracts namespace, command, flags, and arguments from `argv`:

```typescript
import { parseArgs } from '@vertz/cli-runtime';

const parsed = parseArgs(['users', 'get', '--id', '123', '--format', 'json']);
// { namespace: 'users', command: 'get', flags: { id: '123', format: 'json' }, ... }
```

### Parameter resolution

Resolves command parameters from flags, prompts for missing required values, applies defaults:

```typescript
import { resolveParameters } from '@vertz/cli-runtime';

const resolved = await resolveParameters(command, flags, resolvers, context, promptAdapter);
```

### Output formatting

Format API responses as JSON, table, or human-readable text:

```typescript
import { formatOutput } from '@vertz/cli-runtime';

console.log(formatOutput(data, 'table'));

// в”Ӯ id  в”Ӯ name  в”Ӯ role  в”Ӯ
// в”ңв”Җв”Җв”Җв”Җв”Җв”јв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”јв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Ө

// в””в”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

### Authentication

OAuth device code flow and token management:

```typescript
import { createAuthManager } from '@vertz/cli-runtime';

const auth = createAuthManager({ configDir: '~/.myapp' });
await auth.initiateDeviceCodeFlow(client, deviceAuthUrl, clientId);
```

### Help generation

Auto-generates help text for commands and namespaces:

```typescript
import { generateHelp } from '@vertz/cli-runtime';

const help = generateHelp('myapp', '1.0.0', config.commands);
```
